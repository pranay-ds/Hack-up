# Sentinel Frontend (React + Vite)

This directory contains the Sentinel analyst dashboard UI.

## Run Locally

```powershell
npm.cmd install
npm.cmd run dev
```

Default local URL: `http://localhost:5173`

## Build

```powershell
npm.cmd run build
```

## Lint

```powershell
npm.cmd run lint
```

## Deployment

Frontend is intended to deploy to Render static hosting.
